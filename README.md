# ❤️ Streamlit Heart Predictor

A simple **Streamlit web app** that uses a **K-Nearest Neighbors (KNN) model** to predict the risk of heart disease based on patient data.  
This project demonstrates how machine learning can be applied to healthcare datasets for risk prediction.

---

## 📂 Project Structure
- `heart.csv` → Dataset containing patient records
- `predict.py` → Script to train and evaluate the KNN model
- `heart_knn_model.pkl` → Saved trained model
- `app.py` → Streamlit app for interactive predictions
